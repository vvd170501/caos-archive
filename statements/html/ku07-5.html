Problem ku07-5: kr07-5 (дореш)

В аргументах командной строки процессу передаются имена программ для запуска cmd1, cmd2, cmd3, ..., cmdN. Реализовать запуск процессов по схеме, аналогичной команде shell:

while cmd1 || cmd2
do
   cmd3 &
   # ...
   cmdN &
   wait
   # ...
   wait
done

То есть цикл выполняется, пока истинно (в терминах shell) выражение cmd1 || cmd2. На каждой итерации цикла параллельно на выполнение запускаются команды cmd3 ... cmdN, затем родительский процесс ждет их завершения, затем переходит на следующую итерацию цикла.

Команды на выполнение запускать с помощью execlp.

Родительский процесс дожидается завершения всех процессов в цикле, то есть завершения цикла, затем выводит на стандартный поток вывода общее число процессов, выполнение которых на всех итерациях цикла (включая процессы из условия цикла) завершилось успешно. Родительский процесс завершается с кодом 0.
