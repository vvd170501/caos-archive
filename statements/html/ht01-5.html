Real time limit: 5 s

Problem ht01-5: ht01-5 (optional)

Дан тип структуры: 

struct S
{
    int m1[16];
    int m2[16];
};

На языке ассемблера x86 напишите подпрограмму process с прототипом: 

uint32_t process(struct S *p, int i, int j);

Подпрограмма должна вернуть значение 

(p->m1[i] + p->m2[j]) / 2

При вычислении переполнение и выходы за границы массива не проверять.
