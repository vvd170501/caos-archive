Problem sm11-1: unix/files/write-be-16-1

На стандартном потоке ввода подаются 16-битные беззнаковые целые числа в десятичном текстовом представлении. В аргументах командной строки задается имя выходного файла. В выходной файл запишите введенные числа в бинарном виде в порядке байт Big-Endian.

Выходной файл должен быть создан с правами только на чтение и запись только для владельца, если он не существовал. Если файл существовал, права доступа не меняйте.

Для записи выходного файла используйте POSIX API.

Для записи чисел используйте системный вызов write , в который передается буфер размера 2.

Программа не должна использовать библиотечные функции преобразования порядка байт в слове. Не допускается использование низкоуровневых манипуляций с памятью (в том числе преобразование типов указателей, union). Программа должна корректно работать как на процессорах с порядком байт Little-Endian, так и на процессорах с порядком байт Big-Endian.

Все входные данные корректны, все системные вызовы завершаются успешно.
